# Usage
## Requirements
* Hugo
* NPM
  * postcss-cli (`npm install -g postcss-cli`)
  * autoprefixer (`npm install -g autoprefixer`)

## Building
1. `npm install`
1. run hugo
  * for production simply run `hugo`
  * for local development run `HUGO_ENV=development hugo server`
